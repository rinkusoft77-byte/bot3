# 🐝 Bee Bot — Образовательный Telegram-бот о пчёлах

Telegram-бот на русском языке, который в понятной и интерактивной форме рассказывает о мире пчёл.

## Технологии

- **Node.js** (CommonJS, >= 18)
- **Telegraf** v4 — фреймворк для Telegram ботов
- **dotenv** — управление конфигурацией
- **Long polling** по умолчанию (с возможностью переключения на webhook)

## Быстрый старт

### 1. Клонируйте проект

```bash
git clone <your-repo-url>
cd bees_project
```

### 2. Установите зависимости

```bash
npm install
```

### 3. Настройте `.env`

Скопируйте шаблон и укажите токен бота:

```bash
cp .env.example .env
```

Откройте `.env` и замените `your_bot_token_here`:

```
BOT_TOKEN=123456789:ABCdefGHI...
NODE_ENV=development
```

> Получить токен можно у [@BotFather](https://t.me/BotFather) — команда `/newbot`.

### 4. Запустите бота

```bash
# Запуск в production-режиме
npm start

# Запуск в режиме разработки (с автоперезагрузкой)
npm run dev
```

## Переход на Webhook

По умолчанию бот работает в режиме **long polling**.

Для переключения на webhook в `src/index.js` замените:

```js
bot.launch()
```

на:

```js
bot.launch({
  webhook: {
    domain: 'https://your-domain.com',
    port: 3000,
  },
});
```
